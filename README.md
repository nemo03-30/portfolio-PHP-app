# PHP掲示板アプリ

## 概要

PHPとMySQLを用いて作成した掲示板アプリです。

名前、投稿内容、パスワードの登録ができ、投稿内容の表示、パスワードと投稿番号を用いた編集・削除を行うことができます。

## 使用技術

* PHP
* MySQL
* HTML

## 主な機能

* 投稿機能
* 投稿一覧表示機能
* パスワード判定機能
* 投稿編集機能
* 投稿削除機能
* データベースを利用したデータ管理

## 工夫した点

* データベースを利用し、投稿内容を永続的に保存できるようにしました。
* 編集対象の投稿を識別し、既存の内容を更新できるよう実装しました。
* 削除機能では、対象の投稿のみを安全に削除できるようにしました。
* 編集・削除時にパスワード認証を導入し、投稿者本人のみが操作できる仕組みを実装しました。

## ファイル構成

* `bulletin-board.php` : 掲示板のメイン処理
* `create_table.php` : テーブル作成用ファイル
* `drop_table.php` : テーブル削除用ファイル

## 学んだこと

この開発を通じて、PHPとMySQLを連携させたWebアプリケーションの基本的な仕組みや、CRUD処理の実装方法について学びました。また、パスワード認証機能を実装することで、ユーザーの操作を制限する仕組みについて理解を深めることができました。
